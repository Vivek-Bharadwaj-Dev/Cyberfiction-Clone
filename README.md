# Cyberfiction Clone

**🔗 Live Preview:** [https://cyberfiction-work.vercel.app/](https://cyberfiction-work.vercel.app/)

A visually stunning, high-performance web clone of Cyberfiction.io built with HTML, CSS, JavaScript, and GSAP.

## ✨ Features

- **Immersive 3D Experience**: Uses HTML5 Canvas for rendering beautiful fluid visuals.
- **Smooth Scrolling & Animations**: Powered by GSAP and ScrollTrigger for buttery smooth animations tied to user scroll.
- **Fully Responsive**: Carefully crafted design that adapts seamlessly to desktop, tablet, and mobile viewing.
- **Modern UI/UX**: Sleek typography, minimalistic layout, and an aesthetic inspired by futuristic cyber themes.

## 🚀 Technologies Used

- **HTML5** & **Vanilla CSS**
- **Vanilla JavaScript**
- **GSAP (GreenSock Animation Platform)** for core animations
- **ScrollTrigger** for scroll-based animations

## 💻 Local Setup

Since this is a static site without a build system, running it locally is incredibly simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vivek-Bharadwaj-Dev/Cyberfiction-Clone.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd Cyberfiction-Clone
   ```

3. **Run a local development server:**
   - **Using VS Code:** Right-click `index.html` and select **"Open with Live Server"**.
   - **Using Node.js:** `npx serve .`
   - **Using Python:** `python -m http.server`

4. **Enjoy the magic!** Open the given localhost port in your browser.
